Computer Science

Concepts + trace skills + writing clean pseudocode.

What earns marks

  • Correct definitions (data structures, complexity, networks)
  • Tracing algorithms without skipping steps
  • Readable pseudocode (clear logic)
  • Explaining trade-offs (speed vs memory, etc.)

High-impact tips

  • Practice tracing daily (short)
  • Memorize core data structures + when to use them
  • Write “model answers” for definitions
  • Do mixed practice sets weekly
