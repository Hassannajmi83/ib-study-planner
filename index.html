How to use this site

  • Start with Subjects to learn paper structure + common traps.
  • Use Planner to generate a weekly plan you’ll actually follow.
  • Use Study Quiz to match your method to your subjects.
  • Use Resources as a curated “legal links only” library.

Quick Start

1) Open Subjects → pick your hardest course.
2) Write 5 mistakes you keep making.
3) Do 1 timed set this week and fix those mistakes.
