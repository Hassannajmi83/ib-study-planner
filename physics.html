Physics

Physics is marks for method. Clear diagrams + correct assumptions = free marks.

What Physics marks reward

  • Correct setup (diagram, knowns/unknowns, assumptions)
  • Correct equations (not random formula hunting)
  • Method marks (show steps, not just final answer)
  • Units + significant figures + reasonableness checks

Fastest improvement

  • Do problems daily (small but consistent)
  • After marking: write the correct method in your own words
  • Track your top 5 repeated error types
  • Train common patterns (projectiles, circuits, energy, waves)

Papers (how they feel)

Paper 1A (MCQ)

  • Fast reasoning + quick calculations
  • Trap: mixing up direction/sign conventions
  • Strategy: estimate first, then compute

Paper 1B (data-based / skills)

  • Interpretation of graphs/experiments
  • Trap: reading the wrong axis / missing gradient meaning
  • Strategy: label what slope/intercept represent physically

Paper 2 (structured problems)

  • Multi-step solutions, explanations
  • Trap: skipping diagrams and losing method marks
  • Strategy: draw first, then equation, then substitute

HL extras (if applicable)

  • More depth + sometimes additional paper structure
  • Trap: trying to memorize instead of deriving
  • Strategy: learn core derivations and when they apply

Problem-solving method (do this every time)

Step-by-step

  • 1) Draw diagram + define directions
  • 2) List knowns/unknowns with units
  • 3) Choose principle (energy? forces? momentum?)
  • 4) Write equation before substituting
  • 5) Solve algebra cleanly
  • 6) Check units + reasonableness

When to use which idea

  • Forces: when acceleration/dynamics matter
  • Energy: when height/speed conversion is key
  • Momentum: collisions/explosions/impulses
  • Waves: relationship between f, λ, v

Common mark traps

Math/sign traps

  • Wrong sign for acceleration/velocity
  • Using degrees vs radians incorrectly
  • Forgetting to square-root at the end
  • Dropping units mid-solution

Concept traps

  • Mixing speed vs velocity
  • Confusing net force with a single force
  • Assuming frictionless without being told
  • Not stating assumptions (constant acceleration, negligible air resistance)

Study plan (Physics = reps)

Weekly minimum

  • 3 small problem sets (10–25 min)
  • 1 mixed/timed set (20–40 min)
  • 1 session: redo errors only
  • Update Error Log every time

Best marking routine

  • Circle where you first went wrong
  • Rewrite the correct setup (diagram + equation)
  • Write a “fix rule” for next time
  • Redo the same question 2 days later

IA advice (easy wins)

  • Pick a measurable variable with repeated trials.
  • Graph properly: labeled axes + units + uncertainty where possible.
  • Discuss limitations realistically (reaction time, sensor resolution, friction).
  • Improvements must be specific (better sensor, more trials, controlled environment).
