Chemistry

Paper structure + how to study + how to avoid losing easy marks.

What you’re being tested on

  • Knowledge + correct chemical language (keywords)
  • Calculation accuracy (units, sig figs, rearranging)
  • Data/graph interpretation (what the numbers actually mean)
  • Application (predicting, explaining, justifying using theory)

Fastest way to improve marks

  • Do exam-style questions, not rereading notes
  • Write the exact markscheme-style wording
  • Keep an Error Log (same mistakes repeat)
  • Redo the same question set 48 hours later

Papers (what each one feels like)

Paper 1A (MCQ)

  • Speed + recognition + small calculations
  • Common trap: rushing and misreading “increase/decrease”
  • Strategy: eliminate options, then compute/check units

Paper 1B (data-based / skills)

  • Interpreting unfamiliar data/experiments
  • Common trap: describing instead of explaining
  • Strategy: quote the data (numbers/trends) then explain

Paper 2 (structured written)

  • Multi-step calculations + short explanations
  • Common trap: skipping steps = lost method marks
  • Strategy: write the formula, substitute, show units

Paper 3 (if your syllabus has it)

  • Often practical/data skills + option content
  • Common trap: vague lab method / no controls
  • Strategy: identify IV/DV/controls + measurement detail

Study plan (the real one that works)

Weekly routine (minimum)

  • 2 topic sessions (learn + small set)
  • 1 mixed set (older topics mixed together)
  • 1 timed mini-set (10–20 min)
  • Update Error Log after each set

How to do a topic session

  • 5 min: quick recall/blurting (what you remember)
  • 15–25 min: practice questions only
  • 10 min: markscheme review + rewrite model answers
  • 2 min: log mistakes + “fix rule”

Common mark traps (easy marks you lose)

Calculation traps

  • No units shown
  • Wrong sig figs / rounding too early
  • Forgetting to convert (kPa→Pa, cm³→dm³, etc.)
  • Wrong formula rearrangement

Explanation traps

  • Saying “more reactive” without mechanism/why
  • Not referencing electronegativity / bonding / IMFs
  • Not linking to collision theory / equilibrium shift logic
  • Using vague words: “it changes” instead of direction + reason

High-value cheat sheets to build

Organic mechanisms (1 page)

  • Functional groups + key reagents
  • Curly arrow patterns (nucleophile → electrophile)
  • Common intermediates + conditions (UV, heat, catalysts)
  • Oxidation/reduction summary (primary/secondary alcohols etc.)

Equilibrium/energetics summary (1 page)

  • Le Chatelier: change → shift → effect on K
  • ΔH sign + exo/endo interpretation
  • Rate vs equilibrium (not the same thing)
  • Graph meanings (rate curves, Maxwell-Boltzmann)

IA advice (quick but real)

  • Pick a simple question with measurable variables.
  • Do repeats (reliability) + control variables clearly.
  • Uncertainty: show it in processing and graphs.
  • Evaluation = limitations + realistic improvements (not magic).
